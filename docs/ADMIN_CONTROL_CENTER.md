# Центр управління системою

## Посилання для входу

**URL:** `/admin/login`

**Повний URL:** `https://xbase.online/admin/login` (або ваш домен)

## Функціонал блокування акаунтів

### В центрі управління

1. Перейдіть на вкладку **"Бізнеси"**
2. У таблиці знайдіть потрібний бізнес
3. Натисніть кнопку **"Заблокувати"** (червона кнопка)
4. Введіть причину блокування (необов'язково)
5. Підтвердіть блокування

### Функції

- **Блокування окремого акаунту** - кнопка "Заблокувати" в таблиці
- **Розблоковування** - кнопка "Розблокувати" для заблокованих акаунтів
- **Масове блокування** - виберіть кілька акаунтів чекбоксами та виконайте дію
- **Причина блокування** - зберігається в налаштуваннях бізнесу

### Відображення

- **ID колонка** - показує 5-значний `businessIdentifier` (наприклад: 56836)
- **Статус** - "Активний" (зелений) або "Неактивний" (червоний)
- **Дії** - кнопки "Деталі", "Заблокувати"/"Розблокувати"

## API Endpoints

### Блокування через API

```bash
POST /api/business/block
{
  "businessIdentifier": "56836",
  "isActive": false,
  "reason": "Порушення правил"
}
```

### Перевірка статусу

```bash
GET /api/business/block?businessIdentifier=56836
```

## Командний рядок

```bash
# Заблокувати
npm run business:block 56836 "Причина блокування"

# Розблокувати
npm run business:unblock 56836
```

## Доступ

Для входу в центр управління потрібен:
- Email адміністратора
- Пароль адміністратора

Створіть адміністратора через seed або API `/api/admin/admins`
