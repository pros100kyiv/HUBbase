# Налаштування домену xbase.online для Vercel

## Крок 1: Додати домен в Vercel

1. Відкрийте ваш проект на Vercel: https://vercel.com
2. Перейдіть до налаштувань проекту (Settings)
3. Виберіть вкладку "Domains"
4. Натисніть "Add Domain"
5. Введіть `xbase.online` та натисніть "Add"

## Крок 2: Налаштування DNS в CityHost

У вас є два варіанти:

### Варіант A: Використання NS серверів Vercel (рекомендовано)

1. У панелі CityHost знайдіть "NS сервери" (NS servers)
2. Натисніть на іконку редагування (олівець)
3. Замініть поточні NS сервери на:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ```
4. Збережіть зміни
5. Зачекайте 24-48 годин для поширення DNS

### Варіант B: Використання DNS записів (якщо хочете залишити NS сервери CityHost)

1. У панелі CityHost знайдіть "Редагування DNS (A, MX, TXT, CNAME)"
2. Додайте наступні записи:

   **Для основного домену (xbase.online):**
   - Тип: `A`
   - Ім'я: `@` або залишити порожнім
   - Значення: `76.76.21.21`

   **Для www піддомену (www.xbase.online):**
   - Тип: `CNAME`
   - Ім'я: `www`
   - Значення: `cname.vercel-dns.com`

3. Збережіть зміни
4. Зачекайте 24-48 годин для поширення DNS

## Крок 3: Перевірка в Vercel

1. Після додавання домену в Vercel, перевірте статус
2. Vercel покаже статус домену:
   - ⏳ "Pending" - очікується налаштування DNS
   - ✅ "Valid" - домен налаштовано правильно
   - ❌ "Invalid" - потрібно перевірити DNS записи

## Крок 4: Очікування поширення DNS

- Зміни DNS можуть зайняти від кількох хвилин до 48 годин
- Ви можете перевірити статус за допомогою: https://dnschecker.org
- Введіть `xbase.online` та перевірте, чи поширилися зміни по всьому світу

## Важливі примітки:

1. **NS сервери Vercel (Варіант A)** - найпростіший спосіб, Vercel автоматично керує всіма DNS записами
2. **DNS записи (Варіант B)** - якщо вам потрібно керувати іншими DNS записами (наприклад, пошта)
3. Після зміни NS серверів або DNS записів, поштові записи (MX) можуть перестати працювати, якщо вони не налаштовані в Vercel

## Перевірка підключення:

Після налаштування, перевірте:
- Відкрийте `https://xbase.online` - має відкритися ваш сайт
- Відкрийте `https://www.xbase.online` - також має працювати

## Підтримка:

Якщо виникнуть проблеми:
- Перевірте статус в Vercel Dashboard
- Перевірте DNS записи через https://dnschecker.org
- Зверніться до підтримки Vercel або CityHost
